Advanced Product Author for WooCommerce is a plugin that allows the admin to change the authorship of WooCommerce products.

== Description ==

Allows admin to change the authorship of the WooCommerce products!

This plugin allows store owners to change the authorship of the WooCommerce products based on the current users & user roles.

WooCommerce has no option for changing product ownership. This plugin will help you if you want to change WooCommerce author for WooCommerce products.

== üíÅ Who is this for? ==

Advanced Product Author for WooCommerce is specially designed for the store owner who is working with the multiple authors.

If you are running a WooCommerce store that carries multiple authors and wants to change the author of your products. Then, this plugin is for you!

== Key features of Advanced Product Author for WooCommerce ==

<ul>
<li>Easy to change auther of WooCommerce products</li>
<li>Enable/Disable product author selection feature</li>
<li>Change author based on the current users</li>
<li>Change author based on the current user roles</li>
</ul>

== Installation ==

* Download the plugin
* Upload the folder "advanced-product-author-for-woocommerce" to wp-content/plugins (or upload a zip through the Wordpress admin)
* Activate and enjoy!

== Frequently Asked Questions ==

= Q. Do I need to do anything after Installation? =
ANS: No, just install and activate. Enjoy. But if you want to restrict authership option based on the users or user roles or want to add author filter on your shop page then go to the plugin settings page and customize your options.
